O que é HTML

(voltar)

HTML, que significa HyperText Markup Language é a principal linguagem usada para criar e estruturar páginas na web. Ele fornece a base sobre a qual as páginas da web são construídas e exibidas nos navegadores.

Estrutura Básica

(voltar)

HTML usa tags para definir elementos na página. Por exemplo, 'h1" define um cabeçalho de nível 1, 'p' define um parágrafo, e 'a' define um link.

Elementos e Atributos

(voltar)

Um elemento HTML é composto por uma tag de abertura, conteúdo e uma tag de fechamento. Por exemplo: "p>Este é um parágrafo/p" . Tags podem ter atributos que fornecem informações adicionais, como "a href="https://example.com">Link/a>./p>/font size>"

Hierarquia e Aninhamento

(voltar)

HTML permite aninhar elementos dentro de outros elementos para criar uma estrutura hierárquica. Por exemplo, listas (ul ou ol) podem conter itens de lista (li).p>font size>

Semântica

(voltar)

HTML5 introduziu novas tags semânticas, como "header", "footer", "article", e "section", para ajudar a descrever a estrutura e o significado do conteúdo.

Compatibilidade com CSS e JavaScript

(voltar)

HTML é frequentemente usado em conjunto com CSS (Cascading Style Sheets) para estilizar páginas e com JavaScript para adicionar interatividade.

Documentos HTML

(voltar)

Um documento HTML geralmente começa com uma declaração do tipo de documento e é estruturado com as tags "html", "head", e "body". O "head" contém metadados e links para recursos como CSS, enquanto o "body" contém o conteúdo visível da página.
